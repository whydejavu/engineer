## IDE（JAVA开发工具集）
在上一节的时候也介绍了，可以使用文本编辑器+javac编译方式编译HelloWord.java文件，但是真正开发的时候使用VI等文本编辑器是不太好使的，这个时候就需要我们用IDE开发工具来做具体的开发了。现在比较流行的java开发工具有eclipse,idea。eclipse是开源免费的，idea是收费的，也有免费的基础版本（功能会有删减），后面有比较好的破解方式。

### eclipse
Eclipse是著名的**跨平台开源**集成开发环境（IDE）。最初主要用来Java语言开发，目前亦有人通过插件使其作为C++、Python、PHP等其他语言的开发工具。

Eclipse的本身只是一个框架平台，但是众多插件的支持，使得Eclipse拥有较佳的灵活性，所以许多软件开发商以Eclipse为框架开发自己的IDE。

### IDEA

IntelliJ IDEA是一种商业化销售的Java集成开发环境（Integrated Development Environment，IDE）工具软件，由JetBrains软件公司(前称为IntelliJ)发展，提供Apache 2.0开放式授权的社区版本以及专有软件的商业版本，开发者可选择其所需来下载使用。
