## JDK(Java Development Kit)
Java开发工具包，是针对Java开发员的产品，是整个Java的核心，包括了Java运行环境
JRE、Java工具和Java基础类库。
## JRE(Java Runtime Environment)
是运行JAVA程序所必须的环境的集合，包含JVM标准实现及Java核心类库。
## JVM(JVM是Java Virtual Machine)
Java虚拟机的缩写，是整个java实现跨平台的最核心的部分，能够
运行以Java语言写作的软件程序。
