# 1.DOS常用命令

## 总结
现在的电脑已经换成了MAC了，这是因为前几年做云计算和Linux系统打交道比较多，而MAC电脑就是Unix电脑,更方便习惯Linux环境。DOS磁盘操作系统，是以前WINDOWS系统的操作系统，现在保留了其功能。总体上来说linux和dos命令比较的相似，大多数功能是相同的，使用方式也类似。详见参考资源-附录DOS和Linux常用命令对比。

## 常用
```
dir (directory) ：列出当前目录下的文件以及文件夹
md (make directory)： 创建目录
rd (remove directory)：删除目录
cd (change directory)：进入指定目录
cd.. : 退回到上一级目录
cd\ : 退回到根目录
del：删除文件
exit：退出dos命令行
cls (clear screen): 清屏
```
